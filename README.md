# Whiteboard

Simple drawing app

## Demo

You can have a play with it here: https://thepearson.github.io/whiteboard/

## Installing it

```
npm install
```

## Running it

Checkout the repo then run

```
npm run watch
```

Navigate to https://localhost:9000 and profit.

## Building it

```
npm run build
```

## Testing it

```
npm run test
```
